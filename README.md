### Game Start

> ##### 자신이 소유한 카드 중에서 속성 값이 가장 높은 값을 기준으로 등급과 속성값으로 이길 확률 조절

##### 등급별 확률 정의

- **NOMAL 등급**: 이길 확률 30%
- **RARE 등급**: 이길 확률 30% + (자신의 카드 속성/10) => 이길 확률 평균 50~60%
- **UNIQUE 등급**: 이길 확률 30% + (자신의 카드 속성/10) + (자신의 카드 속성/10)/5 => 이길확률 약 60~76%

##### 이길시 보상

- **자신이 투자한 금액을 보상, 지면 자신이 투자한 만큼 손해**
